# Contributing

Thank you for your interest in contributing to Awesome Typography! This document provides guidelines for contributing.

## How to Contribute

### Adding a Font

1. Choose the appropriate category:
   - **Monospaced**: Fonts with fixed-width characters
   - **Sans Serif**: Sans-serif fonts (use `[Grotesk]` label for grotesque fonts)
   - **Experimental**: Unique or experimental typefaces

2. Format your entry:
   ```markdown
   - [Font Name](https://link-to-font.com/) [Free] [Grotesk]
   ```
   - Use `[Free]` for free/open-source fonts (e.g., Google Fonts)
   - Use `[Grotesk]` for grotesque sans-serif fonts
   - Links should point directly to the font's page

3. Maintain alphabetical order within each category

4. Ensure the font:
   - Works well for modern interfaces and web design
   - Is contemporary and well-designed
   - Has a proper website/page

### Adding a Foundry

1. Format your entry:
   ```markdown
   - [Foundry Name](https://foundry-website.com/) [Open Source]
   ```
   - Use `[Open Source]` if the foundry releases open-source fonts
   - Links should point to the foundry's main website

2. Maintain alphabetical order

3. Ensure the foundry:
   - Produces quality contemporary typefaces
   - Has an active website

## Guidelines

- Keep descriptions neutral and factual
- Verify all links work before submitting
- Follow the existing formatting style
- One font/foundry per line
- Use proper capitalization for font and foundry names

## [Open Source]

This project welcomes contributions from the open-source community. All contributions should follow the guidelines above and maintain the quality and organization of the list.
